# Dataset Description

This project uses fitness tracker data to analyze user activity levels, sleep patterns, and health indicators.

# Activity Dataset
Includes daily activity metrics such as:
- Total steps
- Calories burned
- Active and sedentary minutes
- Activity intensity levels

# Sleep Dataset
Contains sleep-related information:
- Total sleep duration
- Sleep patterns and consistency
- Sleep efficiency indicators

# Weight Dataset
Includes health-related measurements:
- Body weight
- Body Mass Index (BMI)
- Health profile variations among users

# Data Processing
- Data was imported and analyzed using Python (Pandas)
- Missing values and duplicates were handled
- Date formats were standardized
- Relevant columns were selected for analysis

# Final Dataset
- Cleaned and processed datasets used for visualization and analysis
- Combined insights from activity, sleep, and weight data
- Used to generate dashboards and business recommendations
